# Coding Tool Trigger Keywords

Use these terms in prompts, descriptions, or examples to improve discovery in coding assistants.

## Tool Names

- Cursor
- Codex
- Claude Code
- OpenCode
- GitHub Copilot

## Task Intent Keywords

- instrument existing agent
- add tracing
- observability for agent runs
- tool call visibility
- missing run timeline
- missing spans
- broken parent/child traces
- token usage not showing
- coding tool telemetry
- prefactor
- prefactor sdk
- withSpan

## Symptom Keywords

- spans missing
- trace tree broken
- tool execution not visible
- final run status missing
- stream finished but span still open
- errors not captured in traces

## Good Trigger Sentence Patterns

- "Instrument this existing agent so Cursor can show run + tool timelines."
- "Add Prefactor SDK tracing without rewriting orchestration logic."
- "Codex cannot see tool calls; add run/tool/llm spans with parent linkage."
- "Claude Code shows no failure trace; capture errors and rethrow originals."
